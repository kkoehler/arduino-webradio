Arduino-based Webradio
----------------------

Implementation for a Arduino Nano based Webradio

Used Parts
----------

* Arduino Nano
* ENC28J60 Microchip HR911105A  (Ethernet)
* VS1053B VS1053 MP3 Modul (Sound)

Used Libraries
--------------

```
#include <EtherCard.h>
#include <net.h>
#include <enc28j60.h>
#include <VS1053.h>
```
